# Frontend Next.js - ERP R&D

## Installation

```bash
npm install
```

## Configuration

Créer un fichier `.env.local` avec:

```
NEXT_PUBLIC_API_URL=http://localhost:8080
```

## Développement

```bash
npm run dev
```

Ouvrir [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
npm start
```
